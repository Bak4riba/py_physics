# **Python e Física**

## **Objetivo deste projeto**

Meu intuito com esse projeto é aplicar python a problemas de física, começando de forma gradual com problemas simples e assim ir aumentando sua complexidade.

## Objetivos
### Objetivo Geral

 - Desenvolver competências em programação científica com Python, aplicando-as à resolução e simulação de problemas clássicos da Física.

### Objetivos Específicos

- Consolidar os fundamentos da linguagem Python

- Aplicar estruturas computacionais à modelagem matemática de fenômenos físicos

- Utilizar bibliotecas científicas para análise numérica e visualização de dados

- Desenvolver raciocínio lógico e computacional aplicado à Física

- Construir um material de estudo sistematizado e progressivo
